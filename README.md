# Personal Finance Tracker and Budgeting App

This project is a personal finance management application.
It helps users track expenses and manage budgets.
The application is designed to be simple and easy to use.
It supports both frontend and backend components.
The backend is implemented using Spring Boot.
The frontend provides a basic and clean user interface.

## Project Purpose

The main goal of this project is to help users
understand their spending habits.
Users can record daily expenses.
They can categorize expenses easily.
The system helps in budget planning.
It also provides basic financial insights.

## Features

Expense tracking functionality
Budget creation and monitoring
Simple and responsive UI
Backend REST APIs
Data management using backend services

## AI Features

AI-based Needs vs Wants classification
Smart analysis of user spending data
Basic AI logic for financial insights
Helps users understand unnecessary expenses
Supports better financial decision making
AI features are integrated into the backend
AI results can be shown in the frontend

## Technologies Used

Java with Spring Boot for backend
HTML for frontend structure
CSS for styling
JavaScript for interactivity
GitHub for version control
